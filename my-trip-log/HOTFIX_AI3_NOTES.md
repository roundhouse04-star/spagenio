# 🩹 Hotfix AI 일정 v3 — 긴 답변도 저장 쉽게

## 추가 수정 (이전 aifix2.zip + 더)

### 문제: 답변이 길면 "일정으로 불러오기" 버튼이 안 보임

**원인**:
1. textarea가 `minHeight: 280`이라 내용 길어지면 **무한히 늘어남**
2. textarea 아래 버튼이 화면 밖으로 밀려나감 (스크롤 필요)
3. 사용자는 끝까지 스크롤 안 하고 버튼 못 찾음
4. "✨ 일정으로 불러오기" 라벨이 모호 (저장이 아니라 "기존 것을 불러옴" 처럼 들림)

### 수정 (4가지)

1. **헤더 우측에 "저장" 버튼 추가** (step 2일 때만)
   - 답변 텍스트 한 글자도 없으면 비활성 (회색)
   - 한 글자라도 있으면 활성화 (골드 색상)
   - **답변이 아무리 길어도 화면 위에 항상 보임** ✨

2. **textarea 고정 높이** (`height: 280, maxHeight: 280`)
   - 내용이 길면 textarea 내부 스크롤
   - 외부 ScrollView가 비정상적으로 길어지지 않음

3. **하단 버튼 라벨 변경**
   - Before: "✨ 일정으로 불러오기" (모호)
   - After: "💾 일정 저장하기" (명확)

4. **글자수 표시**
   - "AI 답변 (1234자)" 처럼 표시
   - 사용자가 붙여넣기 잘 됐는지 즉시 확인

### Before vs After

**Before:**
```
[답변 붙여넣기 화면]
  - textarea가 내용 길어지면 화면 가득
  - 끝까지 스크롤해야 버튼 보임
  - 일부 사용자: 버튼 못 찾음 → 답답함
```

**After:**
```
[답변 붙여넣기 화면]
  - 헤더 우측 "저장" 버튼 항상 보임
  - textarea는 280px로 고정 (내부 스크롤)
  - 하단 "💾 일정 저장하기" 도 함께 표시 (선택)
  - 어디든 눌러도 동일 동작
```

## 테스트
1. AI 일정 → 답변 붙여넣기 화면
2. ✅ 헤더 우측 "저장" 회색 (비활성)
3. 답변 짧게 입력 → ✅ "저장" 골드로 변함
4. 답변 매우 길게 입력 (10000자) → ✅ "저장" 여전히 보임
5. textarea 안에서 위/아래 스크롤 가능
6. 헤더 "저장" 또는 하단 "💾 일정 저장하기" 어느 쪽 눌러도 동일하게 작동
