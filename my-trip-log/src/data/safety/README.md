# Triplive 여행 안전 데이터 (1.2)

## 📋 데이터 분류

### 정적 데이터 (이 디렉토리, 앱 번들 포함)
- **embassies.ts**: 도시별 대사관/영사관 위치 + 연락처
- **emergencyGuides.ts**: 비상 상황 대처법 (여권 분실, 도난, 부상 등)
- **diseases.ts**: 국가별 권장 백신 + 일반 감염병 정보 (자주 안 바뀜)

### 동적 데이터 (Cloudflare Workers + D1, Phase 2)
- **여행 경보**: 외교부 API (data.go.kr)
- **안전공지**: 외교부 API (실시간)
- **감염병 발생**: WHO API (실시간 변동)
- **사용자 신고**: D1 (소매치기 위험 지역 등)

### 외부 API 출처
| API | 인증 | 비용 | 갱신 |
|---|---|---|---|
| 외교부 여행경보단계 | data.go.kr 인증키 | 무료 | 일간 |
| 외교부 안전공지 | data.go.kr 인증키 | 무료 | 실시간 |
| WHO 감염병 | 인증 X | 무료 | 일~주간 |
