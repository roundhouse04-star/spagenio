# 이용약관

**최종 업데이트: 2026년 4월 22일**

본 약관은 「내공연관리」(이하 "본 앱") 의 이용에 관한 조건을 정합니다. 본 앱을 이용함으로써 이용자는 본 약관에 동의한 것으로 간주됩니다.

## 제1조 (목적)

본 약관은 본 앱의 개발자(이하 "개발자") 가 제공하는 서비스의 이용 조건과 방법, 이용자와 개발자의 권리·의무 등을 규정합니다.

## 제2조 (정의)

1. "본 앱" 이란 개발자가 제공하는 iOS / Android 모바일 애플리케이션 「내공연관리」를 의미합니다.
2. "이용자" 란 본 앱을 다운로드·설치·실행하는 자연인을 의미합니다.
3. "콘텐츠" 란 본 앱을 통해 제공되는 공연 정보, 아티스트 정보, 이미지, 텍스트 등 일체의 정보를 의미합니다.

## 제3조 (서비스 내용)

본 앱은 다음 기능을 제공합니다.

1. 아티스트 검색 및 팔로우 관리
2. 공연 일정 조회 및 알림
3. 관람 티켓 기록 (사진 포함)
4. 데이터 백업·복원 (JSON 내보내기/가져오기)
5. 외부 공개 데이터 (KOPIS · 위키백과) 를 활용한 정보 제공

## 제4조 (서비스의 무료 제공)

본 앱은 현재 무료로 제공되며, 유료 기능이 추가될 경우 사전에 고지합니다.

## 제5조 (이용자의 의무)

이용자는 다음 행위를 하여서는 안 됩니다.

1. 본 앱을 리버스 엔지니어링·디컴파일·변조하는 행위
2. 본 앱의 API 또는 서버에 대해 비정상적 자동 요청을 보내는 행위
3. 타인의 저작권·초상권·명예를 침해하는 이미지나 텍스트를 티켓 정보로 입력하여 제3자에게 배포하는 행위
4. 본 앱을 상업적 데이터 수집·재판매 목적으로 이용하는 행위
5. 기타 관계 법령에 위반되는 행위

## 제6조 (데이터의 정확성 및 면책)

1. 본 앱이 표시하는 공연 정보는 **공연예술통합전산망(KOPIS)** 과 **위키백과** 등 외부 데이터 출처의 정보를 재가공하여 제공합니다.

2. 원 데이터의 변경·오류·지연으로 인해 실제 공연 일정·장소·가격 등과 다를 수 있습니다. **이용자는 실제 예매·관람 전 반드시 공식 판매처 또는 주최자 웹사이트에서 정보를 재확인해야 합니다.**

3. 개발자는 원 데이터 출처의 정보 정확성·최신성·완결성에 대해 보증하지 않으며, 이로 인해 발생한 이용자의 손해에 대해 책임지지 않습니다.

4. 본 앱은 공연 티켓의 예매·결제 기능을 제공하지 않으며, 제3자의 티켓 판매 페이지로의 링크만 제공합니다. 해당 페이지의 거래는 전적으로 이용자와 해당 판매자 간의 관계이며, 개발자는 어떠한 책임도 지지 않습니다.

## 제7조 (저작권 및 라이선스)

1. 본 앱 자체의 코드 · 디자인 · 상표의 저작권은 개발자에게 귀속됩니다.

2. 본 앱이 표시하는 외부 출처 콘텐츠의 저작권은 각 원 저작권자에게 귀속되며, 다음 라이선스에 따라 이용됩니다.

   - **KOPIS 공연 정보**: 공공데이터 (공공데이터의 제공 및 이용 활성화에 관한 법률에 따른 이용)
   - **위키백과 텍스트**: CC BY-SA 4.0 (Creative Commons Attribution-ShareAlike 4.0)

3. 이용자가 본 앱에서 확인한 위키백과 출처 콘텐츠를 외부로 재배포할 경우, CC BY-SA 4.0 의 저작자 표시 및 동일 조건 변경 허락 의무를 준수해야 합니다.

4. 이용자가 본 앱에 직접 입력하는 티켓 이미지·메모 등의 저작권은 이용자 본인에게 귀속됩니다. 이 데이터는 개발자의 서버로 전송되지 않고 이용자 기기에만 저장됩니다.

## 제8조 (서비스의 변경·중단)

1. 개발자는 기술적 필요성, 외부 API 의 정책 변경, 법령 준수 등의 사유로 본 앱의 기능 일부 또는 전부를 변경·중단할 수 있습니다.

2. 주요 변경 사항은 앱 업데이트 릴리즈 노트 또는 앱 내 공지를 통해 고지합니다.

3. KOPIS·위키백과 등 외부 API 의 중단·정책 변경으로 인한 서비스 제한에 대해 개발자는 책임지지 않습니다.

## 제9조 (개인정보 보호)

이용자의 개인정보 처리에 관한 사항은 별도의 **[개인정보처리방침]** 에 따릅니다. 본 앱은 이용자의 데이터를 개발자 서버로 전송하지 않으며, 모든 데이터는 이용자 기기에만 저장됩니다.

## 제10조 (면책)

1. 천재지변, 전쟁, 기간통신사업자의 서비스 중단, 외부 API 제공자의 장애 등 개발자의 합리적 통제를 벗어난 사유로 인한 서비스 이용 제한에 대해 개발자는 책임지지 않습니다.

2. 본 앱은 "있는 그대로(AS-IS)" 제공되며, 상업적 가치, 특정 목적 적합성, 비침해성에 대한 명시적·묵시적 보증을 제공하지 않습니다.

3. 이용자의 기기 고장·저장공간 부족·운영체제 업데이트 등으로 인한 데이터 손실에 대해 개발자는 책임지지 않습니다. **이용자는 설정 → "데이터 내보내기" 기능을 통해 주기적으로 백업할 것을 권장합니다.**

## 제11조 (준거법 및 관할)

1. 본 약관의 해석 및 이용자와 개발자 간의 분쟁에 대하여는 대한민국 법을 준거법으로 합니다.

2. 본 약관과 관련하여 발생한 분쟁은 민사소송법상의 관할법원에 제기합니다.

## 제12조 (약관의 변경)

개발자는 필요한 경우 본 약관을 변경할 수 있으며, 변경된 약관은 앱 업데이트 시 앱 내 "앱 정보" 화면에 게시됩니다. 이용자가 변경된 약관 게시 후에도 본 앱을 계속 이용하는 경우 변경 약관에 동의한 것으로 간주됩니다.
