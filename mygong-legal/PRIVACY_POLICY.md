# 개인정보처리방침

**최종 업데이트: 2026년 4월 22일**

「내공연관리」(이하 "본 앱")은 이용자의 개인정보를 소중하게 생각하며, 「개인정보 보호법」 등 관련 법령을 준수하고 있습니다. 본 방침은 본 앱이 어떤 정보를 수집하고, 어떻게 이용하는지를 투명하게 안내하기 위해 마련되었습니다.

## 1. 수집하는 정보와 저장 방식

### 1-1. 본 앱은 이용자의 개인정보를 **서버로 전송하거나 수집하지 않습니다.**

본 앱은 서비스 제공자(개발자)의 서버가 없으며, 이용자가 앱 내에서 생성·입력·저장하는 모든 정보는 **이용자 본인의 기기에만 저장**됩니다.

기기에 저장되는 정보의 예시:
- 팔로우한 아티스트 목록 · 수동 입력한 공연 정보 · 티켓 기록
- 티켓 이미지 (카메라 촬영 또는 사진 라이브러리에서 선택한 경우)
- 알림 설정값 · 온보딩 완료 여부

### 1-2. 외부 API 호출 과정에서의 정보

본 앱은 공연·아티스트 정보를 가져오기 위해 다음 외부 API 를 호출합니다. 호출 과정에서 이용자의 **검색어**와 **기기의 IP 주소**가 해당 서비스에 전달될 수 있으며, 이는 각 서비스의 개인정보처리방침에 따릅니다.

| 외부 API | 제공자 | 전달되는 정보 | 개인정보처리방침 |
|----------|--------|---------------|------------------|
| mygong-api (Cloudflare Worker) | 본 앱 개발자 | 검색 조건(공연명·기간·장르), IP | ※ 로깅 비활성화 권장 |
| KOPIS Open API | 문화체육관광부 · 예술경영지원센터 | 검색 조건, IP | https://www.kopis.or.kr |
| Wikipedia API | Wikimedia Foundation | 검색어, IP, User-Agent | https://foundation.wikimedia.org/wiki/Privacy_policy |

**본 앱은 이용자의 이름, 이메일, 전화번호, 주소, 결제 정보 등 일체의 식별 가능한 개인정보를 요구·수집·저장하지 않습니다.**

## 2. 정보의 이용 목적

이용자의 기기에 저장된 정보는 오직 본 앱의 기능(공연 일정 관리·알림·티켓 기록) 제공을 위해서만 이용됩니다. 개발자는 이 정보에 접근할 수 없습니다.

## 3. 정보의 보관 기간 및 파기

- 기기 내 정보는 이용자가 삭제하거나 앱을 제거할 때까지 보관됩니다.
- 설정 화면의 **"모든 데이터 삭제"** 기능으로 언제든지 전체 삭제가 가능합니다.
- 앱 제거 시 모든 정보는 영구 파기됩니다.

## 4. 정보의 제3자 제공

본 앱은 이용자의 정보를 제3자에게 제공하지 않습니다. (애초에 개발자가 정보를 수집하지 않기 때문입니다.)

## 5. 이용자의 권리

이용자는 본인의 기기에 저장된 모든 정보에 대해 다음 권리를 가집니다.

- **열람 권리**: 앱의 각 화면에서 본인의 데이터를 확인할 수 있습니다.
- **수정 권리**: 앱 내 편집 기능을 통해 언제든지 수정할 수 있습니다.
- **삭제 권리**: 설정 → "모든 데이터 삭제" 또는 앱 제거로 전체 삭제 가능합니다.
- **이전 권리**: 설정 → "데이터 내보내기" 로 JSON 백업 파일을 만들어 다른 기기로 옮길 수 있습니다.

## 6. 기기 권한 사용

본 앱은 다음 권한을 필요 시에만 요청하며, 거부하더라도 핵심 기능은 이용할 수 있습니다.

| 권한 | 용도 | 거부 시 영향 |
|------|------|--------------|
| 사진 라이브러리 접근 | 티켓 사진 추가 | 갤러리에서 사진 선택 불가 (직접 촬영은 가능) |
| 카메라 접근 | 티켓 사진 촬영 | 앱 내 촬영 불가 (갤러리 선택은 가능) |
| 알림 (푸시) | 공연 D-day 알림 | 알림 받을 수 없음 |

이 권한은 본 앱이 이용자 기기 내에서만 사용하며, 외부로 전송하지 않습니다.

## 7. 쿠키 및 추적 기술

본 앱은 광고 추적, 행동 분석, 제3자 광고 SDK 를 **일체 사용하지 않습니다.**

## 8. 만 14세 미만 이용자

본 앱은 만 14세 미만의 개인정보를 고의로 수집하지 않습니다. 앱 내에 수집·저장되는 모든 정보는 이용자 본인의 기기에만 존재합니다.

## 9. 정보 보호 조치

본 앱은 이용자의 정보를 외부로 전송하지 않지만, 기기 내 데이터 보호를 위해 다음 조치를 취합니다.

- **로컬 SQLite 데이터베이스**에만 저장하며, 외부 클라우드 동기화 없음
- **KOPIS API 키**는 앱 소스에 포함되지 않고, 중간 서버(Cloudflare Worker) 의 암호화된 환경변수에만 저장

## 10. 개인정보처리방침의 변경

본 방침이 변경되는 경우, 앱 내 "앱 정보" 화면을 통해 고지합니다. 중요한 변경 사항은 별도 공지합니다.
